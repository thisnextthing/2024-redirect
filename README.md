# 2024.thisnextthing.com redirect

Serves path-preserving redirects from the retired `2024.thisnextthing.com`
subdomain to the archived site at
[thisnextthing.com/2024](https://thisnextthing.com/2024/), via GitHub Pages.

`index.html` handles the root; `404.html` catches every other path and
forwards it to the matching page in the archive.
